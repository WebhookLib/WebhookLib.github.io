Introduction

WebhookLib is a production-quality Discord webhook library designed specifically for Roblox server environments. It provides a comprehensive set of features for sending Discord messages with automatic rate limiting, error handling, and rich embed support.

Why WebhookLib?

WebhookLib addresses common challenges when working with Discord webhooks in Roblox:

  • Rate Limiting: Automatic request throttling to prevent hitting Discord's rate limits
  • Error Handling: Robust error handling with automatic retries and fallback mechanisms
  • Player Tracking: Built-in player join/leave notifications with avatar thumbnails
  • Rich Embeds: Easy-to-use embed builder with automatic text sanitization
  • Thread Safety: Non-blocking operations that won't freeze your game

Installation

There are several ways to install WebhookLib in your Roblox game:

Method 1: Roblox Model

Download the WebhookLib model from the Roblox catalog and insert it into your game.

Steps 
1. Open Roblox Studio
2. Go to the Toolbox
3. Search for "WebhookLib"
4. Insert the model into ReplicatedStorage or ServerStorage

Method 2: Copy and Paste

Copy the WebhookLib module script and paste it into your game:

Lua 
-- Create a ModuleScript in ReplicatedStorage named "WebhookLib"
-- Then paste the WebhookLib code inside

Directory Structure

Place WebhookLib in one of these locations:

  • game.ReplicatedStorage.WebhookLib (Recommended)
  • game.ServerStorage.WebhookLib
  • game.ServerScriptService.WebhookLib

Quick Start

Get up and running with WebhookLib in just a few minutes:

Step 1: Create a Discord Webhook

First, create a webhook in your Discord server:

  1. Right-click on the channel where you want to receive messages
  2. Select "Edit Channel" → "Integrations" → "Webhooks"
  3. Click "Create Webhook"
  4. Copy the webhook URL

Step 2: Basic Setup

Lua 
local WebhookLib = require(game.ReplicatedStorage.WebhookLib)

-- Create a new webhook instance
local webhook = WebhookLib.new("YOUR_WEBHOOK_URL_HERE")

-- Send your first message
webhook:SendMessage("Hello from Roblox!")

Step 3: Player Notifications

Lua 
-- Automatic player join notifications
game.Players.PlayerAdded:Connect(function(player)
    webhook:SendJoinMessage(player)
end)

-- Automatic player leave notifications
game.Players.PlayerRemoving:Connect(function(player)
    webhook:SendLeaveMessage(player)
end)

Step 4: Rich Embeds

Lua 
-- Send a rich embed
webhook:SendEmbed({
    title = "Server Statistics",
    description = "Current server information",
    color = 0x00ff00,
    fields = {
        {
            name = "Players Online",
            value = tostring(#game.Players:GetPlayers()),
            inline = true
        },
        {
            name = "Server Uptime",
            value = "2 hours 30 minutes",
            inline = true
        }
    },
    timestamp = os.date("!%Y-%m-%dT%H:%M:%SZ")
})
