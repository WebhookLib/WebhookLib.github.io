WebhookLib Production-Quality Discord Webhook Library

A comprehensive Discord webhook library specifically designed for Roblox Server environments. Features automatic rate limiting, player tracking, rich embeds, and robust error handling.

Get Started View Examples
Lua 
local WebhookLib = require(game.ReplicatedStorage.WebhookLib)

-- Create a new webhook
local webhook = WebhookLib.new("YOUR_WEBHOOK_URL")

-- Send a simple message
webhook:SendMessage("Hello, Discord!")

-- Send a rich embed
webhook:SendEmbed({
    title = "Game Event",
    description = "A player joined the game!",
    color = 0x00ff00,
    fields = {
        {
            name = "Player",
            value = "ExamplePlayer",
            inline = true
        }
    }
})

Powerful Features

Production Ready

Built for high-traffic games with robust error handling, automatic retries, and comprehensive logging capabilities.

Rate Limiting

Optional automatic rate limiting with request queueing and exponential backoff to prevent Discord API limits.

Player Tracking

Automatic join/leave notifications with Roblox avatar thumbnails and persistent join count tracking via DataStore.

Rich Embeds

Send beautiful Discord embeds with automatic text sanitization, field validation, and content filtering.

Thread Safe

Non-blocking operations that won't freeze your game, with thread-safe request handling and memory management.

Comprehensive Logging

Debug mode for troubleshooting with detailed request/response logging and performance monitoring.

Code Examples

Basic Setup

Lua 
local WebhookLib = require(game.ReplicatedStorage.WebhookLib)

-- Create webhook with basic configuration
local webhook = WebhookLib.new("YOUR_WEBHOOK_URL", {
    username = "My Game Bot",
    avatar_url = "https://example.com/avatar.png",
    default_color = 0x0099ff,
    debug = true
})

-- Send a test message
webhook:SendMessage("🎮 Server started successfully!")

Player Events

Lua 
-- Automatic player join/leave notifications
game.Players.PlayerAdded:Connect(function(player)
    pcall(function()
        webhook:SendJoinMessage(player)
    end)
end)

game.Players.PlayerRemoving:Connect(function(player)
    pcall(function()
        webhook:SendLeaveMessage(player)
    end)
end)

Rich Embeds

Lua 
-- Send a rich embed with game statistics
webhook:SendEmbed({
    title = "🏆 Game Statistics",
    description = "Current server statistics",
    color = 0x0099ff,
    fields = {
        {
            name = "👥 Players Online",
            value = tostring(#game.Players:GetPlayers()),
            inline = true
        },
        {
            name = "⏰ Server Uptime",
            value = "2 hours 15 minutes",
            inline = true
        }
    },
    thumbnail = {
        url = "https://example.com/thumbnail.png"
    },
    footer = {
        text = "Automated Report"
    },
    timestamp = os.date("!%Y-%m-%dT%H:%M:%SZ")
})

Advanced Configuration

Lua 
-- Production-ready setup with rate limiting
local webhook = WebhookLib.new("YOUR_WEBHOOK_URL", {
    username = "🤖 Production Bot",
    queue_rate_limit = 2, -- 2 requests per second
    max_retries = 5,
    retry_backoff_base = 2,
    filter_profanity = true,
    cache_ttl_avatar = 600, -- 10 minutes
    datastore_name = "MyGame_WebhookData",
    banned_words = {"spam", "advertisement"}
})
